# Matrix Transpose in C

This program takes a matrix from the user and calculates its transpose (swapping rows and columns).

## Features
- Supports up to 10x10 matrices.
- User-friendly input/output.
- Basic boundary checks for stability.

## How to Compile and Run
1. Open your terminal or IDE.
2. Compile the `main.c` file.
3. Run the generated executable.
